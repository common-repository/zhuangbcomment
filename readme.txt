== Description ==

Insert comments in your post with [zhuangb event="mouseover" id="1" title="Leave a reply"]~

== Installation ==

Upload `zhuangb` to the `/wp-content/plugins/` directory
Activate the plugin through the 'Plugins' menu in WordPress
Place `[zhuangb event="mouseover" id="1" title="Leave a reply"]` in your post
